Modules¶

Almost all functionality provided by Python Startkladde is implemented in several modules. These modules are distributed over several packages for different purposes.

All modules are summarised in the main package called pysk (for python startkladde). This module contains the following sub-modules:

  • The db package for database operations
  • Tools package which contains the implementation of most command line tools
  • utils/utils
